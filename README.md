# 💻 Laptop Price Prediction with Machine Learning

This project focuses on predicting laptop prices using multiple machine learning regression algorithms.

## Project Overview

The dataset contains laptop specifications such as:

* RAM
* CPU Type
* CPU Frequency
* Weight
* Screen Size
* GPU Information
* Operating System

The main goal of the project is to compare different machine learning regression models and evaluate their performance in laptop price prediction.

## Machine Learning Models Used

* Linear Regression
* K-Nearest Neighbors (KNN)
* Random Forest Regressor

## Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Google Colab

## Project Steps

1. Data loading and exploration
2. Data cleaning
3. Feature encoding
4. Train-test split
5. Model training
6. Performance evaluation
7. Visualization and comparison of results

## Model Results

| Model                   | MAE      | R² Score |
| ----------------------- | -------- | -------- |
| Linear Regression       | 314.69 € | 0.66     |
| KNN Regressor           | 237.31 € | 0.77     |
| Random Forest Regressor | 221.97 € | 0.80     |

The Random Forest model achieved the best overall performance.
